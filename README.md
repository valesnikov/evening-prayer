# XeLaTeX Project

## Структура

- `src/main.tex` — основной файл документа
- `src/content.tex` — контент документа
- `assets/fonts/` — шрифты
- `build/` — результат сборки и временные файлы

## Сборка

```bash
./build.sh
```

Итоговый PDF:

- `build/molitvoslov.pdf`

## Очистка

```bash
./clean.sh
```
